# post-app

## How start project on Windows:
```
git clone https://github.com/LuckyMen000/fullstack-app

cd fullstack-app

pip -r requirements.txt

python manage.py runserver
```
## For Linux and Mac OS 

```
git clone https://github.com/LuckyMen000/fullstack-app

cd fullstack-app

pip3 -r requirements.txt

python3 manage.py runserver
```
